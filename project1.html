Smart Home Energy Monitor

This project involved designing and building an IoT-based energy monitoring system for residential applications. Using an ESP32 microcontroller, current sensors, and a cloud dashboard, the system tracks real-time power consumption of household appliances.

Technical Details

  • ESP32 microcontroller with Wi-Fi connectivity
  • Non-invasive current transformers (CTs) for safe monitoring
  • MQTT protocol for data transmission
  • Node-RED dashboard for visualization
  • Data logging to InfluxDB for analysis

Results

The system achieved ±2% accuracy in power measurement and identified opportunities for reducing standby power consumption by 15% in the test home. This project won first place in the Pitt Engineering Design Showcase (Fall 2025).
