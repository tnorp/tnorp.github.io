Long-COVID Appointment Tracker

The Problem

When my mom was diagnosed with long-COVID, she suddenly faced a overwhelming maze of medical appointments. Between primary care physicians, pulmonologists, neurologists, cardiologists, and specialists, keeping track of appointments, medications, and symptoms became a full-time job—one that added stress during an already difficult time.

The Solution

I built a centralized appointment tracking system designed specifically for patients managing multiple specialists. The tracker consolidates all appointments in one place, sends reminders, and provides a clear overview of the care journey—reducing cognitive load and ensuring no critical appointment is missed.

Feature Highlight: The tracker sends SMS reminders 24 hours before each appointment, stores doctors, operates in two visual modes, tracks recurring appointments, and has address lookup.

Technical Implementation

  • Frontend: HTML5, CSS3, JavaScript with responsive design for mobile access
  • Backend: Node.js with Express for API endpoints
  • Database: PostgreSQL for secure appointment and user data storage
  • Reminders: Twilio API for SMS notifications

Key Features

  • Centralized dashboard showing all upcoming appointments across specialists
  • SMS reminder system to prevent missed appointments
  • Mobile-friendly design for on-the-go access

Impact & Results

Since implementing the tracker, my mother has:

  • 0 missed appointments in the past 3 months
  • Reported significantly reduced stress around managing her care
  • Shared the system with her long-COVID support group
